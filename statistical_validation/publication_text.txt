METHODS SECTION TEXT:
--------------------------------------------------------------------------------

Hyperparameter Sensitivity Analysis

To assess the importance of data-driven hyperparameter optimization, we compared the GridSearch-optimized baseline against four manually-selected parameter configurations: (1) loose threshold (tau=0.05, lambda=10), (2) strict threshold (tau=0.30, lambda=10), (3) weak penalty (tau=0.15, lambda=5), and (4) strong penalty (tau=0.15, lambda=50). Each manual configuration was applied uniformly across all spots, in contrast to the GridSearch baseline which optimized parameters independently for each spot using 5-fold cross-validation (repeated 3 times) over a grid of tau=[0.001-0.10] and lambda=[0.001-10.0].

Statistical significance was assessed using the Wilcoxon signed-rank test (paired, non-parametric, two-sided) with alpha=0.05. This test was chosen for its robustness to non-normal distributions and outliers, which are common in biological data. Effect sizes were quantified using Cohen's d (parametric) and Cliff's delta (non-parametric), where |d| < 0.2 indicates negligible effect, 0.2-0.5 small, 0.5-0.8 medium, and >0.8 large effect. All statistical analyses were performed in Python 3.13 using SciPy 1.11.


RESULTS SECTION TEXT:
--------------------------------------------------------------------------------

Data-driven hyperparameter optimization is essential for WISpR deconvolution

Manual hyperparameter selection consistently underperformed compared to GridSearch optimization across all tested configurations. The Loose τ (0.05) configuration (tau=0.05, lambda=10) achieved a mean RMSE of 5.242, representing a 91.3% increase in reconstruction error compared to the GridSearch baseline (mean RMSE = 0.454; Wilcoxon signed-rank test p < 1e-100, Cohen's d = -2.64). The Strict τ (0.30) configuration (tau=0.3, lambda=10) achieved a mean RMSE of 4.840, representing a 90.6% increase in reconstruction error compared to the GridSearch baseline (mean RMSE = 0.454; Wilcoxon signed-rank test p < 1e-100, Cohen's d = -2.41). The Weak λ (5) configuration (tau=0.15, lambda=5) achieved a mean RMSE of 10.505, representing a 95.7% increase in reconstruction error compared to the GridSearch baseline (mean RMSE = 0.454; Wilcoxon signed-rank test p < 1e-100, Cohen's d = -2.76). The Strong λ (50) configuration (tau=0.15, lambda=50) achieved a mean RMSE of 0.779, representing a 41.6% increase in reconstruction error compared to the GridSearch baseline (mean RMSE = 0.454; Wilcoxon signed-rank test p < 1e-100, Cohen's d = -0.77). 

All manual configurations showed statistically significant increases in reconstruction error (all p < 0.001), with effect sizes ranging from -2.76 to -0.77, indicating substantial practical differences beyond mere statistical significance. 

Importantly, manual configurations also produced biologically implausible sparsity patterns. While the GridSearch baseline achieved a mean of 4.2 cell types per spot (consistent with expected cellular complexity in Visium data), manual configurations predicted 26.0 (Loose τ (0.05)), 20.0 (Strict τ (0.30)), 26.0 (Weak λ (5)), 12.6 (Strong λ (50)) cell types per spot, far exceeding biological expectations. 

These results demonstrate that data-driven hyperparameter optimization via GridSearch is not merely beneficial but essential for achieving accurate, biologically meaningful deconvolution with WISpR. The spot-specific optimization strategy allows WISpR to adapt to local tissue heterogeneity, which cannot be captured by uniform manual parameter selection.
