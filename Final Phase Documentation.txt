some info about the .loom file
The .loom file you downloaded contains:
- scRNA-seq data from mouse hippocampus (L1 = Level 1 hierarchy in the Linnarsson atlas)
- Gene expression counts (genes × cells)
- Cell type annotations (already classified by the Linnarsson Lab)
- Metadata (cluster information, cell types, QC metrics)

A .loom file is an HDF5-based format designed for large single-cell datasets. It contains:
- Layers: Expression matrices (raw counts, normalized counts)
- Row attributes: Gene metadata (gene names, IDs)
- Column attributes: Cell metadata (cell types, clusters, QC metrics)
This is more compact than separate mtx/barcodes/features files, but we'll need to extract the data into a format WISpR can use.

#pause to understand - DEG Selection:
- Problem: We have 16,390 genes, but most are "housekeeping genes" (expressed everywhere equally) - not useful for distinguishing cell types.
- Solution: Use Gini coefficient to find genes that are cell-type-specific:
	High Gini (0.9+): Gene expressed strongly in ONE cell type, weakly in others → GOOD for identifying that cell type
	Low Gini (0.3-): Gene expressed equally everywhere → USELESS for deconvolution
- What we selected: 2,000 genes with Gini ≥ 0.759 (highly specific markers)
- Example: Slc13a1 has Gini 0.962 - only expressed in Cluster_22_Neurons, so when we see this gene in a spatial spot, we know neurons are there!
- Why this matters: WISpR will use ONLY these 2,000 informative genes (not all 16K) to figure out cell types in each spot.

🎓 INTERPRETING THE RESULTS
What p-values mean:

p < 0.05: Statistically significant (baseline ≠ experiment)
p < 0.01: Highly significant
p < 0.001: Very highly significant (use *** in tables)

What Cohen's d means:

|d| < 0.2: Negligible effect
|d| 0.2-0.5: Small effect
|d| 0.5-0.8: Medium effect
|d| > 0.8: Large effect

For your paper: Negative d means baseline is BETTER (lower RMSE)
What Cliff's delta means:

Same interpretation as Cohen's d, but non-parametric
More robust to outliers
Use this for publication if distributions are skewed
